# 一口气拆穿Skill/MCP/Agent/OpenClaw原理

> 作者：飞天闪客 | 点赞：58990 | 收藏：73494 | 评论：1060

## 正文核心内容

- 这是小红书上最热门的 OpenClaw 原理讲解视频，5.9万赞、7.3万收藏
- 视频系统性拆解了 Skill、MCP、Agent、OpenClaw 四层概念的关系
- 核心逻辑：大模型（LLM）是"大脑"，Skill 是"工具箱"，Agent 是"调度员"，OpenClaw 是"操作系统"
- 把 AI 从只能"聊天"升级到能"做事"的关键在于工具调用（Tool Use）能力的接入

## 评论区补充知识

- Agent 的本质："像极了大厂的PD，把运营的需求翻译给技术听"——@夏甜豆（13赞）
- Token 的通俗解释："AI 使用的信息编码，把文字切片转化为 token，比如'今天'就是 001，'天气'就是 002"——@陀鸟（55赞）
- vibe coding = 用非程序语言完成编程，即用 AI 编程——@MOMO（4赞）
- 问答式 AI（豆包千问）只包括大模型推理部分，OpenClaw 在此基础上加了工具调用和记忆
- 向量数据库的用途：搜索本地文件时，需要将文本向量化，用相似度检索找相关内容
- Memory 机制：评论区有人指出视频"没有讲 memory"，作者回复"讲了"
